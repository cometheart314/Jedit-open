Jedit Pro のプライバシーに関する方針

概要

Jedit Pro はユーザーのプライバシーを尊重します。本アプリは、個人情報の収集、保存、または第三者への送信を行いません。

データの収集について

Jedit Pro は以下のデータを収集しません：

  • 個人を特定できる情報（氏名、メールアドレス等）
  • 位置情報
  • 連絡先やカレンダーのデータ
  • ドキュメントの内容
  • マルチ検索・置換で対象としたファイルの内容や検索文字列

ネットワーク通信

Jedit Pro は起動時にアプリ内メッセージの確認のため、サーバーに接続します。この通信において：

  • 送信されるのは標準的な HTTP リクエストのみです
  • ドキュメントの内容やユーザーの個人情報は送信されません
  • 利用状況の統計のため、アクセス元の IP アドレスをハッシュ化（不可逆変換）した情報を記録します
  • ハッシュ化された情報から元の IP アドレスを復元することはできません
  • この情報は日別のユニークユーザー数の集計にのみ使用されます

iCloud

Jedit Pro は iCloud Drive を使用してドキュメントを同期する機能を提供します。iCloud に保存されるデータは Apple のプライバシーポリシーに従って管理されます。Jedit Pro の開発者が iCloud 上のデータにアクセスすることはありません。

マルチ検索・置換でアクセスするファイル

マルチ検索・置換機能は、ユーザーが明示的に指定したファイルおよびフォルダのみを対象として動作します。これらのファイルは検索・置換のためにローカルで読み込み・書き込みされるだけで、外部に送信されることはありません。検索文字列・置換文字列・履歴は、お使いの Mac のローカルストレージにのみ保存されます。

ローカルデータ

アプリの設定（フォントサイズ、表示設定、検索履歴、プリセット等）はお使いの Mac のローカルストレージに保存され、外部に送信されることはありません。
