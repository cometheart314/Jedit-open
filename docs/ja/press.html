概要

Jedit は、macOS 向けのフリーテキストエディタです。プレーンテキストからリッチテキスト（RTF / RTFD）、Microsoft Word、Markdown まで幅広いフォーマットに対応し、日本語環境での文書作成に最適化されています。かつて多くのユーザーに利用された Jedit Ω を原型とし、誰もが毎日必要とする機能に厳選して再設計された汎用テキストエディタです。

Mac App Store にて 無料で提供され、ソースコードは GitHub にて GPLv3 のもと公開されています。

主な機能

多彩なフォーマット

プレーンテキスト、RTF、RTFD、Microsoft Word、Markdown を単一のエディタで編集。

日本語に最適化

縦書き表示、豊富な日本語エンコーディングの自動判定、CJK タイポグラフィへの配慮。

高度な検索・置換

正規表現対応、マッチハイライト、検索履歴の永続化。

iCloud Drive 統合

複数の Mac 間で書類をシームレスに同期。

執筆支援ツール

文字数・単語数・段落数のリアルタイム統計、執筆目標の設定。

階層ブックマーク

長文構成管理に適した階層型ナビゲーション。

AppleScript 対応

完全な AppleScript 辞書を搭載し、ワークフロー自動化が可能。

印刷機能

ヘッダー・フッター、行番号、カラー印刷などきめ細かな設定。

豊富な表示モード

スプリットビュー、ページ表示、ルーラー、行番号、不可視文字表示。

Jedit Ω との関係

Jedit Ω は、Mac 用の高機能エディタとして多くのユーザーに愛用されてきましたが、コードベースが古くなり、新しい macOS への追従が困難になったため、2025 年 11 月末をもって販売およびサポートを終了しました。

新しい Jedit は、Jedit Ω をベースに、AI（Claude Code）を全面的に採用して全くゼロから書き直された後継エディタです。書き直しにあたっては、一部のユーザーしか使わない機能、稀にしか使われない機能はあえて再実装を見送り、誰もが日常的に必要とする機能に絞って汎用テキストエディタとして再構築しました。

再実装が見送られた Jedit Ω の機能

  • マルチファイル検索 / 置換
  • 文書ファイルの比較機能
  • Jedit Ω ツールメニューの各種機能
  • シンタックスカラーリング
  • スマートインデックス、自動補完
  • メニューショートカットのカスタマイズ
  • キーバインドのカスタマイズ

これらの機能は、今後 Jedit Pro としてサポートが予定されています。
