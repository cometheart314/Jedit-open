Jedit Pro

プロ向け機能を備えた macOS テキストエディタ

Jedit Pro について

Jedit Pro は、無料の Jedit に旧版 Jedit Ω 由来の高度な編集機能を加えた上位版テキストエディタです。Jedit のすべての機能に加えて、複数ファイルにまたがる検索・置換、強制改行、ソート、Unicode 正規化など、長文・大量ファイルを扱う方や日本語処理を多用する方向けの機能を搭載しています。

Jedit との違い

Jedit Pro は Jedit が備える基本機能（複数フォーマット対応、横書き/縦書き、iCloud 同期、AppleScript、印刷機能、文字コード対応など）をすべて含みます。その上で、以下の Pro 専用機能が利用できます。

Pro 専用の主な機能

マルチ検索・置換

複数のファイルを横断して検索し、結果パネルからジャンプ・一括置換が可能。マルチペアの一括置換にも対応します。

ツールメニュー

テキスト処理に役立つコマンドを多数搭載。下記の各種機能をワンクリックで実行できます。

強制改行

指定した幅で行を折り返して LF を挿入。プレフィックスの付加・除去や、自動インデント保持にも対応。

ソート

行・単語単位のソート。降順、数値、日本語読み順、半角/全角・大文字/小文字・欧文補助記号の無視など細かく制御できます。

Unicode 正規化

NFC・NFD・NFKC・NFKD の各形式に対応。Mac/Windows 間のテキスト互換性確保や、文字コード問題の解決に役立ちます。

漢字部首の検出と変換

「⼈」「⽇」など部首単独文字を検出し、対応する標準 CJK 文字（人・日 など）に一括変換します。

半角⇔全角変換

英数字、カタカナ、記号それぞれの半角/全角を細かく指定して変換。プリセット保存機能付き。

整形・クリーンアップ

行頭/行末スペースの削除、タブ/スペースインデントの相互変換、重複行の削除・カウント、行の逆順並べ替えなど。

その他のツール

URL 文字列への自動リンク付加、ひらがな⇔カタカナ変換、漢字→ひらがな変換、文字コード表示など。

今後追加予定の機能

  • キーバインドのカスタマイズ
