Jedit

macOS 用の高機能テキストエディタ

Jedit について

Jedit は、macOS 向けに開発されたフリーのテキストエディタです。プレーンテキストからリッチテキスト（RTF/RTFD）まで幅広いフォーマットに対応し、日本語環境での文書作成に最適化されています。

Mac App Store でダウンロード（日本） Mac App Store（US） GitHub でソースコードを見る

スクリーンショット

主な機能

多彩なフォーマット

プレーンテキスト、RTF、RTFD、Word 文書、Markdown に対応。用途に応じて最適なフォーマットで作業できます。

豊富な表現機能

横書き/縦書き、ページ表示、スプリットビュー、行番号表示、ルーラー表示などを搭載。

高度な検索・置換

正規表現対応の検索・置換機能。検索履歴の管理やマッチのハイライト表示も搭載。

iCloud 同期

iCloud Drive との完全な統合。複数の Mac 間でドキュメントをシームレスに同期できます。

執筆支援ツール

文字数・単語数・段落数のリアルタイム統計。執筆目標の設定と進捗の追跡も可能。

ブックマーク

階層的なブックマークでドキュメント内をすばやくナビゲート。長文の構成管理に便利です。

AppleScript 対応

完全な AppleScript 辞書を搭載。テキスト処理の自動化やワークフローの構築が可能です。

印刷機能

ヘッダー・フッター、行番号印刷、カラー印刷など、きめ細かな印刷オプションを用意。

文字コード対応

多数の文字エンコーディングに対応。文字コードの自動検出と変換機能を搭載しています。

旧版 Jedit Ω と Jedit Pro について

Jedit Ω は、Mac 用の高機能エディタとして多くのユーザーに愛用されてきましたが、コードベースが古くなり、新しい macOS への追従が困難になったため、2025 年 11 月末をもって販売およびサポートを終了しました。

新しい Jedit は、Jedit Ω をベースに、AI（Claude Code）を全面的に採用して全くゼロから書き直された後継エディタです。書き直しにあたっては、一部のユーザーしか使わない機能や稀にしか使われない機能はあえて再実装を見送り、誰もが日常的に利用する必要にして十分な機能を備えた汎用テキストエディタをめざしました。

Jedit でサポートしていない Jedit Ω の機能

  • マルチファイル検索/置換機能
  • 文書ファイルの比較機能
  • Jedit Ω ツールメニューの各種機能
  • シンタックスカラーリング
  • スマートインデックス、自動補完
  • メニューショートカットのカスタマイズ
  • キーバインドのカスタマイズ

Jedit Pro

Jedit に含まれていない機能の一部は、上位版の Jedit Pro でサポートしています。マルチファイル検索・置換、強制改行、ソート、Unicode 正規化、漢字部首検出などのツール群を搭載し、Jedit Ω からの移行ユーザーや日常的に大量のテキストを扱う方に向けた構成です。
