Overview

Jedit is a free text editor for macOS. It supports a wide range of formats — from plain text to rich text (RTF / RTFD), Microsoft Word, and Markdown — and is optimized for authoring in both Japanese and English. Built on the lineage of Jedit Ω, a long-standing favorite among Japanese Mac users, Jedit is a general-purpose editor redesigned around the essentials that anyone needs on a daily basis.

Jedit is available free of charge on the Mac App Store, and the source code is published on GitHub under the GPLv3 license.

Key Features

Multiple Formats

Edit plain text, RTF, RTFD, Microsoft Word, and Markdown in a single editor.

Optimized for Japanese

Vertical text layout, automatic detection of many Japanese encodings, careful CJK typography.

Advanced Find & Replace

Regular-expression search and replace, match highlighting, persistent search history.

iCloud Drive Integration

Seamlessly sync documents across multiple Macs.

Writing Tools

Real-time character, word, and paragraph counts; configurable writing goals.

Hierarchical Bookmarks

Hierarchical navigation well suited to long-form documents.

AppleScript Support

Complete AppleScript dictionary for workflow automation.

Print Features

Fine-grained control over headers, footers, line numbers, and color printing.

Rich Display Modes

Split view, page display, rulers, line numbers, and invisible-character display.

Relationship to Jedit Ω

Jedit Ω was loved by many users as a powerful text editor for the Mac, but its codebase had grown old and could no longer keep up with newer versions of macOS. For that reason, sales and support for Jedit Ω ended at the end of November 2025.

The new Jedit is a successor editor rewritten from scratch, with AI (Claude Code) adopted across the entire development process, using Jedit Ω as a reference. As part of this rewrite, we intentionally chose not to reimplement features that only a small subset of users used or that were rarely needed, and refocused the product as a general-purpose text editor centered on the features everyone uses daily.

Jedit Ω features not carried over to Jedit

  • Multi-file find / replace
  • Document file comparison
  • Various Jedit Ω Tools menu features
  • Syntax coloring
  • Smart indentation and auto-completion
  • Menu shortcut customization
  • Key-binding customization

These features are planned for a future Jedit Pro offering.
