Jedit

A powerful text editor for macOS

About Jedit

Jedit is a free text editor designed for macOS. It supports a wide range of formats from plain text to rich text (RTF/RTFD), and is optimized for document creation in both Japanese and English environments.

Key Features

Multiple Formats

Supports plain text, RTF, RTFD, Word documents, and Markdown. Work in the format that best suits your needs.

Rich Expressive Features

Equipped with horizontal/vertical writing, page display, split view, line number display, ruler display, and more.

Advanced Find & Replace

Powerful find and replace with regular expression support, search history management, and match highlighting.

iCloud Sync

Full iCloud Drive integration. Seamlessly sync your documents across multiple Macs.

Writing Tools

Real-time statistics for characters, words, and paragraphs. Set writing goals and track your progress.

Bookmarks

Navigate documents quickly with hierarchical bookmarks. Perfect for managing the structure of long documents.

AppleScript Support

Complete AppleScript dictionary for automating text processing and building custom workflows.

Print Features

Customizable headers and footers, line number printing, color options, and detailed print settings.

Character Encodings

Supports numerous character encodings with automatic detection and conversion capabilities.

About Legacy Jedit Ω and Jedit Pro

Jedit Ω was loved by many users as a powerful text editor for the Mac, but its codebase had grown old and could no longer keep up with newer versions of macOS. For that reason, sales and support for Jedit Ω ended at the end of November 2025.

The new Jedit is a successor editor rewritten from scratch, with AI (Claude Code) adopted across the entire development process, using Jedit Ω as a reference. As part of this rewrite, features used by only a small number of users or rarely used features were intentionally not reimplemented. Our goal was to create a general-purpose text editor with all the essential features that everyone needs for daily use.

Jedit Ω Features Not Supported in Jedit

  • Multi-file find and replace
  • Document file comparison
  • Various Jedit Ω Tools menu features
  • Syntax coloring
  • Smart index and auto-completion
  • Menu shortcut customization
  • Key binding customization

Jedit Pro

Some of the features not included in Jedit are available in Jedit Pro, the upgraded edition. It includes multi-file find & replace, hard line wrapping, sort, Unicode normalization, kanji radical detection, and other tools — built for users migrating from Jedit Ω or anyone who handles large volumes of text on a daily basis.
