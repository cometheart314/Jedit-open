Privacy Policy

Jedit's commitment to your privacy

Overview

Jedit respects your privacy. This application does not collect, store, or transmit any personal information.

Data Collection

Jedit does not collect:

  • Personally identifiable information (name, email address, etc.)
  • Location data
  • Contacts or calendar data
  • Document contents

Network Communication

Jedit connects to a server at launch to check for in-app messages. During this communication:

  • Only standard HTTP requests are sent
  • No document contents or personal information are transmitted
  • For usage statistics, a hashed (irreversibly transformed) version of the access IP address is recorded
  • The original IP address cannot be recovered from the hashed data
  • This information is used solely to count daily unique users

iCloud

Jedit offers document syncing through iCloud Drive. Data stored in iCloud is managed in accordance with Apple's privacy policy. The developer of Jedit does not have access to any data stored in iCloud.

Local Data

Application preferences (font size, display settings, etc.) are stored on your Mac's local storage and are never transmitted externally.

Contact

For privacy-related inquiries, please visit GitHub Issues.
