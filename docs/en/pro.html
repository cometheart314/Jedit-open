Jedit Pro

A macOS text editor with advanced features for power users

About Jedit Pro

Jedit Pro is the upgraded edition of the free Jedit, with advanced editing features inherited from the legacy Jedit Ω. In addition to everything Jedit offers, it provides multi-file find and replace, hard line wrapping, sorting, Unicode normalization, and many other tools designed for users who handle long documents, large numbers of files, or extensive Japanese text processing.

How It Differs from Jedit

Jedit Pro includes all of Jedit's core features (multiple format support, horizontal/vertical writing, iCloud sync, AppleScript, printing, character encoding support, and more). On top of those, the following Pro-only features are available.

Pro-only Key Features

Multi-File Find & Replace

Search across multiple files at once. Jump from a results panel to any match, replace selected matches in bulk, and run multi-pair batch replacements.

Tools Menu

A wide set of text-processing commands you can run with a single click. The features below are all part of this menu.

Hard Line Wrapping

Wrap lines at a chosen width by inserting LFs. Supports adding/removing prefixes and preserving auto-indent.

Sort

Sort by line or word, with fine-grained options: descending order, numeric sort, Japanese reading order, and ignoring half/full width, case, or diacritic marks.

Unicode Normalization

Convert text to NFC, NFD, NFKC, or NFKD. Useful for ensuring text compatibility between Mac and Windows and for resolving character-encoding issues.

Kanji Radical Detection

Detect standalone CJK radical characters (such as ⼈, ⽇) in your text and convert them to the corresponding standard CJK characters (人, 日, etc.) in bulk.

Half / Full Width Conversion

Convert half-width and full-width forms of letters, katakana, and symbols separately, with fine-grained control. Save your conversion rules as presets.

Cleanup & Reformatting

Remove leading/trailing spaces, convert between tab and space indentation, remove or count duplicate lines, and reverse line order.

Other Tools

Auto-link URL strings, convert between hiragana and katakana, transliterate kanji to hiragana, show character codes, and more.
