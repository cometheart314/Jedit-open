# Press Release

**For Immediate Release**

## Jedit v1.0, a Free Text Editor for macOS, Now Available on the Mac App Store

— A general-purpose text editor inheriting the legacy of Jedit Ω, optimized for Japanese and English document creation —

**Tokyo, April 19, 2026** — **Artman21 Inc.** today announced the release of **Jedit v1.0**, a free text editor for macOS, on the Mac App Store. Jedit is a general-purpose text editor that carries forward the lineage of **Jedit Ω**, a long-standing favorite among Japanese Mac users, while being redesigned around the essential features that any user needs on a daily basis.

### Key Features

Jedit tightly integrates the standard text-processing facilities of macOS to cover a broad range of authoring needs — from plain text to rich text.

- **Multiple format support** — Edit plain text, RTF, RTFD, Microsoft Word (.doc / .docx), and Markdown in a single editor
- **Optimized for Japanese writing** — Vertical text layout, automatic detection of a wide range of Japanese character encodings, and careful handling of CJK typography
- **Advanced find &amp; replace** — Regular-expression search and replace, match highlighting, and persistent search history
- **iCloud Drive integration** — Seamlessly sync documents across multiple Macs
- **Writing tools** — Real-time character, word, and paragraph counts with support for writing goals
- **Hierarchical bookmarks** — Well suited to navigating long-form documents
- **AppleScript support** — Complete AppleScript dictionary for automating workflows
- **Print features** — Fine-grained control over headers and footers, line-number printing, and color printing
- **Split view, page display, ruler, and line-number view**

### Relationship to Jedit Ω

Jedit Ω was loved by many users as a powerful text editor for the Mac, but its codebase had grown old and could no longer keep up with newer versions of macOS. For that reason, **sales and support for Jedit Ω ended at the end of November 2025**. The new Jedit is a successor editor **rewritten from scratch, with AI (Claude Code) adopted across the entire development process**, using Jedit Ω as a reference. As part of this rewrite, features used by only a small subset of users or rarely needed were intentionally not reimplemented, and the product was refocused around the features everyone uses daily. Capabilities that were not carried over — such as multi-file find/replace, file comparison, syntax coloring, smart indentation, auto-completion, and key-binding customization — are planned for a future **Jedit Pro** offering.

### About the Publisher

Jedit is published by **Artman21 Inc.** (合資会社アートマン弐壱), an independent software studio based in Japan, led by representative Satoshi Matsumoto, who also serves as the lead developer of Jedit.
